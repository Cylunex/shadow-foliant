# 🤖 复合多AI智能体股票团队分析系统

> 基于 DeepSeek / OpenAI 兼容大模型的 A股多智能体分析平台:模拟"分析师团队"对个股做技术/基本面/资金/情绪/风险多维研判与决策,并提供选股、盯盘、持仓管理、回测与盈利反馈闭环。

## 🚀 本分支增强(在原版基础上)

在原"多智能体诊股 + 选股 + 盯盘"之上,做了一批面向 **可盈利、可部署、可被 Agent 调用** 的强化:

| 方向 | 增强 |
|---|---|
| **数据源·可移植** | A股取数链 `东财直连 → mootdx(通达信公网) → akshare/tushare → Ashare(零依赖兜底)`;盯盘 TDX 改 mootdx 优先,**不再写死内网服务**,异地/容器可跑 |
| **盈利闭环** | AI 推荐记录入场价、追踪**真实盈亏**(非"触发率")、按 source 真实战绩**回喂选股门槛**;买入**盈亏比≥2 硬约束** |
| **决策质量** | **阶段决策护栏**(盘前/非交易"立即买卖"→改"开盘确认"+降信心,治保守偏置);多智能体 context 喂入 **行情 regime + 缩量回踩/底部放量/情绪顶信号 + 筹码分布(获利盘/成本)** |
| **回测** | **双收益回测**(裸持有 vs 带止损止盈纪律)+ 止损/止盈触发率,验证止损止盈位是否合理 |
| **选股防飞刀** | 候选池加"缩量回踩/底部放量"强确认,纯低位下跌(接飞刀)剔除 |
| **MCP Server** | `mcp_server.py` 暴露 **28 个工具**(数据/缠论/风险/打分/排雷/选股/组合/筹码/信号/双收益回测/多智能体深析 + 妙想东财AI第二意见),供 Claude/OpenClaw 等 Agent 调用,**不依赖 UI** |
| **投资透镜** | 巴菲特/段永平/纳瓦尔/韭菜 + 游资打板/产业链卡位,多视角决策复核 |

## ⚡ 快速开始

```bash
# 1. 依赖(含 mootdx / mcp)
pip install -r requirements.txt

# 2. 配置:复制并填写 .env(至少 DEEPSEEK_API_KEY;生产建议 USE_POSTGRES=true + PG_*)
cp .env.example .env

# 3. 启动 Web 界面(默认端口见 .streamlit/config.toml)
streamlit run app.py

# 4. (可选) 启动 MCP server,供 Agent 接入
python mcp_server.py
```

> Docker 部署见 [BUILD_CN.md](BUILD_CN.md) / `docker-compose.yml`;完整配置、运行、部署、能力清单见 **[使用文档.md](使用文档.md)**。

## 🧩 功能模块

- **股票分析**:多智能体团队(技术/基本面/资金/情绪/风险/新闻)讨论 → 决策(评级/目标价/止盈止损/盈亏比/投资透镜复核)
- **选股板块**:主力选股 / 低价擒牛 / 小市值 / 净利增长 / 低估值 / 多因子横截面 / 选股配方
- **策略分析**:智策板块 · 智瞰龙虎(龙虎榜)· 新闻流量监测 · 宏观分析 / 宏观周期
- **投资管理**:持仓管理 · AI 盯盘 · 实时监测(交易时段自动启停)· 成交导入 · 组合诊断 · 持仓风险日报(VaR/筹码/情绪)
- **后台任务**(jobs_hub):盘前预热 / 盘后快照 / 策略扫描 / 候选池 / 缠论扫描 / 持仓风险 / 周回测 / AI 推荐追踪与评估
- **报表与回测**:报表中心 · 双收益策略回测
