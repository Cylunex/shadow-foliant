# 📈 shadow-foliant — A股多智能体投研平台

> 基于 DeepSeek / OpenAI 兼容大模型的 A股量化投研系统:多智能体"分析师团队"对个股做技术/基本面/资金/情绪/风险多维研判与决策,覆盖**选股 → 盯盘 → 持仓/基金管理 → 回测 → 盈利反馈闭环**,并以 **MCP 工具**暴露全部能力供 Agent(Claude / OpenClaw 等)调用。FastAPI + Vue3 SPA 前端,通知驱动。

## ⚡ 快速开始

```bash
# 1. 依赖
pip install -r requirements.txt

# 2. 配置:复制并填写 .env(至少 DEEPSEEK_API_KEY;生产建议 USE_POSTGRES=true + PG_* / REDIS_*)
cp .env.example .env

# 3. 启动 Web 界面 → http://localhost:8601
python webui/run_dev.py                 # 本机开发(改前端即时生效;改 .py 需重启)
# 生产:uvicorn webui.api_server:app --port 8601 --log-config webui/log_config.json

# 4. (可选) 启动 MCP server,供 Agent 接入(80+ 工具,不依赖 UI)
python mcp_server.py

# 5. (可选) 定时任务常驻:python -m jobs.jobs_hub --serve(或 supervisor 守护)
```

> Docker 部署见 [BUILD_CN.md](BUILD_CN.md);完整配置/运行/能力清单见 **[使用文档.md](使用文档.md)**。

## 🧩 核心能力

- **🏠 股票分析**:多智能体团队(技术/基本面/资金/情绪/风险/新闻)讨论 → 决策(评级/目标价/止盈止损/盈亏比硬约束≥2/投资透镜复核)+ DCF 内在价值估值 + RAG 证据
- **🎯 选股**:多因子横截面 · 主力选股 · 低价擒牛 · 小市值 · 净利增长 · 问财配方 · AI 双层推荐(短线+长期)· **可转债双低**
- **🧬 策略进化**:13 套 InStock 策略全参数化 + 策略基因组三层进化(FIXED/DYNAMIC/COMPOSED),**walk-forward 样本外部署门**(只上线样本外也成立的变体)
- **👁️ 监控盯盘**:进场区间/止盈止损监控 · 实时监测(交易时段自动启停)· 风控门(fail-closed)
- **📊 持仓/基金**:持仓总览(今日盈亏/绩效 TWR·XIRR/压力测试/组合体检 X-Ray/净值曲线)· 成交批量导入 · **组合级回测**(实盘口径:现金账户/并发上限/成本/对比沪深300)· 交易习惯洞察 · 持仓分级 · 加减仓操作信号 · AI 持仓诊断 · 基金定投(回测/三策略对比/估值定投)· **基金对比/组合诊断** · 已实现盈亏
- **📡 市场**:大盘指数 · 板块强弱 · 北向资金 · 龙虎榜 · 财经快讯 · 东财妙想 AI(交易日复盘 + 周末研判)
- **🔁 盈利闭环**:选股 → 推荐池(记入场价)→ 盘中追踪真实盈亏 → 周评估真实胜率 → 回喂选股门槛
- **🎯 决策信号统一层**:每次分析/选股/盯盘/AI 能力的操作主张抽成结构化信号(8 态动作 + 生命周期 + 去重 + 反向作废),**K线方向后验真实命中率**,按 source/动作/周期分桶可比
- **🧠 AI 赋能**:持仓 AI 体检官(单股 持有/减仓/清仓)· 研报增量解读(评级方向/隐含空间)· 组合压力情景叙事(8 情景风险预案)· 选股红蓝对抗证伪(多空裁判)—— **均接决策信号后验,可量化验证增量**
- **🤖 后台任务**:盘后 K线预热/快照/因子采集/策略回测进化(进程池并行)/AI 推荐追踪/基金净值/决策信号后验;**任务失败自动告警(带 traceback)+ 交易日守卫**

## 🏛️ 架构要点

- **统一数据层 `data/datahub.py`**:全项目取外部数据的唯一门面,自适应多源路由(按健康度升降级)+ **三级缓存**(进程内存 → Redis → 本地文件 pickle),削外部调用、优雅降级。
- **后端**:FastAPI(`webui/api_server.py`,~110 路由)· 多数据源兜底(东财/腾讯/新浪/同花顺/akshare/tushare/mootdx)· PG 主库 + SQLite 兜底 · Redis 缓存 · BGE-M3 + pgvector RAG。
- **前端**:Vue3 SPA(`webui/static/`,A股红涨绿跌/中文表头);**手机版 H5**(汉堡抽屉导航)· **PWA**(可装主屏幕,API 不缓存)· 亮/暗主题。
- **Agent 接入**:`mcp_server.py`(FastMCP)暴露行情/分析/选股/回测/持仓/基金/可转债/语义搜索/任务触发等 80+ 工具。
- **通知驱动**:晨报/选股/盯盘/今日盈亏/任务失败 等推送(默认 QQ webhook,可配邮件)。
