## 路径引导(重要)

为保持原有**扁平 import**(`from stock_data import ...`)与**动态 import**(`__import__(name)`)零改动,
分目录后用 `_bootstrap.py` 在进程入口把各子目录加入 `sys.path`:

- **根入口**(app.py/run.py/autostart.py):首行 `import _bootstrap` 即可。
- **scripts/ 或子目录下独立运行的文件**:
  ```python
  import os, sys
  sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
  import _bootstrap  # noqa
  ```
- **需要项目根路径**:`import _bootstrap; _bootstrap.ROOT`(原 `os.path.dirname(__file__)` 定位根 DB 的写法已统一改用 `_bootstrap.ROOT`)。

> 因此**移动文件无需改任何 import 语句**;新增子目录请同步加入 `_bootstrap._SUBDIRS`。

## 数据库路径

所有 SQLite 库统一在 `db/`。代码用 `_bootstrap.db_path('xxx.db')` 取绝对路径
(`= ROOT/db/xxx.db`;`:memory:` 或绝对路径原样返回)。新增 SQLite 库请用此助手,
不要再写相对文件名。.env / 日志 / 配置仍在根目录。
> 注:大部分数据已用 PostgreSQL(`USE_POSTGRES=true`),SQLite 仅作本地缓存/归档/降级。
