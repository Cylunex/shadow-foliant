# 📖 使用文档

> 复合多AI智能体股票团队分析系统 —— 安装 · 配置 · 运行 · 能力 · 部署 一站式说明。
> 仅 A股(沪深京 6 位代码);AI 分析需大模型 API Key。**输出为辅助分析,不构成投资建议。**

---

## 一、环境与安装

- Python 3.10+(开发环境为 3.14)
- 安装依赖:

```bash
pip install -r requirements.txt
```

关键依赖:`streamlit`(Web)、`akshare/tushare/pywencai/mootdx`(数据)、`openai`(大模型,OpenAI 兼容)、`mcp`(MCP server)、`psycopg2-binary`(PG)、`reportlab`(PDF)、`schedule/pytz`(定时)。

> ⚠️ 中文 PDF 报告需系统中文字体:Windows 自带;Linux/Docker 装 `fonts-wqy-zenhei` 或 Noto CJK。

---

## 二、配置(.env)

复制模板并填写:`cp .env.example .env`(Windows:`copy .env.example .env`)。

| 变量 | 必填 | 说明 |
|---|---|---|
| `DEEPSEEK_API_KEY` | ✅ | 大模型 Key(DeepSeek 或任意 OpenAI 兼容) |
| `DEEPSEEK_BASE_URL` | ✅ | LLM API 地址，如方舟 `https://ark.cn-beijing.volces.com/api/coding/v3` |
| `DEFAULT_MODEL_NAME` | ✅ | 模型名，如 `deepseek-v4-pro`/`qwen-plus`/`gpt-4o` |
| `USE_POSTGRES` + `PG_HOST/PG_PORT/PG_DATABASE/PG_USER/PG_PASSWORD` | 生产建议 | 用 PostgreSQL 持久化(持仓/成交/推荐/盈亏闭环);不配则用本地 SQLite(`db/`) |
| `TUSHARE_TOKEN` | 可选 | Tushare 降级数据源 |
| `EM_API_KEY` | 可选 | 妙想(东财 AI SaaS)`mx_*` 工具用;不填用内置 demo key(易限流) |
| `TDX_SERVERS` / `TDX_USE_MOOTDX` / `TDX_HTTP_URL` | 可选 | TDX 数据源:`TDX_SERVERS="ip:port,ip:port"` 指定通达信服务器;`TDX_USE_MOOTDX=true`(默认) 优先公网 mootdx;`TDX_HTTP_URL` 兜底的内网 TDX HTTP 服务地址 |
| 邮件 `SMTP_*` / Webhook `WEBHOOK_*` | 可选 | 监测/定时分析的推送通道 |
| `RATE_LIMIT_*` | 可选 | 外部接口自限流(防封)节流参数 |

> PG 模式首次需建表:执行 `scripts/init_postgres.sql`;盈亏追踪列由程序运行时幂等补齐,无需手动迁移。

## 三、运行

### 1. Web 界面(主入口)
```bash
streamlit run app.py
```
默认端口见 `.streamlit/config.toml`。`app.py` 顶部 `import autostart` 会按 `.env` 开关自动拉起后台服务(监测/调度)。

### 2. 后台定时任务(jobs_hub)
盘前预热、盘后快照、策略扫描、候选池、缠论扫描、持仓风险、周回测、AI 推荐追踪与评估等,由 `jobs/jobs_hub.py` 驱动。
- 任务开关在 Admin UI(「🛠️ AI 管理」)或 `jobs/automation_config.py`。
- ⚠️ **需让 jobs_hub 常驻运行**(Web 进程内有调度线程;独立部署可 `python -m jobs.jobs_hub` 或用 cron/常驻进程拉起)。任务时间按 UTC 注册、用 `schedule` 的 tz 换算到本机时区,迁到任意时区机器都正确。

### 3. MCP Server(供 Agent 调用,可选)
```bash
python mcp_server.py        # stdio 传输
```
MCP 客户端(Claude Code / OpenClaw 等)配置:
```json
{ "command": "python", "args": ["<项目根>/mcp_server.py"],
  "env": { "DEEPSEEK_API_KEY": "...", "USE_POSTGRES": "true", "EM_API_KEY": "..." } }
```

## 四、功能使用(Web · WebUI 13 页)

> 入口 http://localhost:8601。顶部常驻**大盘指数条**(上证/深证/创业板/科创50/沪深300/恒生,30s 刷新,**新浪→腾讯双源兜底**)。所有列表可**点表头排序**。A股惯例红涨绿跌。**手机访问自动切响应式**(汉堡☰ 抽屉导航;可"添加到主屏幕"装成 PWA 全屏 App;右上角切亮/暗主题)。
> **数据源原则**:外部数据都尽量多源可切换,主源挂自动换备源(`_try` helper),全挂才失败。东财 push2 不稳/被墙时走新浪/腾讯/同花顺等。

**懒人入口(默认首页)**
- **☀️ 晨报**:一眼看完——大盘速览 + 今日买入候选(多因子)+ **持仓逐只扫描→该卖的几只(带理由)** + 持仓买点 + 🤖AI一句话操作提示。缓存 2h,点刷新重算。
- **📬 主动推送**:`scripts/push_briefing.py` 每天盘前自动生成晨报推到邮箱(QQ SMTP)。挂 Windows 任务计划即可,不用开页面/后台。配置见 `.env`(`EMAIL_PASSWORD`=QQ邮箱授权码、`NOTIFICATION_ROUTE_REPORT=email`);监控触发也走同一邮件通道。

**个人持仓**
- **📊 持仓总览**:股票持仓表 + 大类资产 + 组合体检(压力测试/净值曲线/X-Ray/绩效 TWR·XIRR) + **🧭交易习惯洞察**(持有时长/买卖频次/最活跃) + **🚦持仓分级**(健康/观察/警报) + **🎯操作信号**(加/减仓) + **🧠AI持仓诊断**。
- **🏦 基金定投**(7 tab):
  - 我的基金:持有表(份额/成本/估值/市值/浮盈)+ 记一笔申赎(移动加权成本)+ 流水;「⟳ 更新净值」刷净值入库(净值读库,秒开)。
  - 查询·回测:单只净值/评分/**📋基金档案**(重仓/评级/经理)/定投回测(普通/估值智能/价值平均,**三策略对比**)+ AI 研判面板。
  - 基金筛选(同类排行,收益/费率过滤)· 指数估值(PE分位+定投倍数)· 定投计划(可启停/删除)· **组合诊断**(配置/集中度/重仓穿透)· **基金对比**(多只并排+叠加净值)。

**分析**
- **🏠 股票分析**:行情 + K线 + 🤖多智能体 AI 深析(技术+基本面+风险→决策,RAG 证据,**📄导出研报MD**)。**个股研究卡**:缠论 / 筹码分布 / 策略信号(regime+缩量回踩/底部放量/情绪顶)/ 财务排雷 / 资金流。**策略回测卡**:裸持有 vs 止损止盈纪律双收益。**💰DCF 内在价值卡**:两阶段折现 + 安全边际 + 敏感性表。

- **🧠 妙想AI**:东财 AI 9 技能(个股/基金诊断、热点、可比、资讯、宏观、行业/主题研报)。

- **🧩 AI工作流**:自定义编排——选数据块 + 配智能体(提示词/模型) + 两层流程(并行分析→综合) → 出结果,存成模板复用。内置 7 模板(个股双层/排雷/多空辩论/盘前/游资题材/持仓体检/基金研判);支持数据预览、运行历史、导出 MD。**新分析逻辑不用写代码,搭即可**。

**行情 / 监测**

## 七、数据源与可靠性

A股取数链(自动多级兜底,任一成功即返回):
**东财 HTTP 直连(a_stock_adapter)→ akshare → tushare → pywencai → Ashare(腾讯/新浪零依赖)/ mootdx(通达信公网)**。

- 外部接口已内置**自限流防封**(`core/rate_limiter.py`),批量调用稍慢属正常,勿并发猛拉。
- mootdx 连公网通达信服务器(`TDX_SERVERS` 可指定);其复权在新版 pandas 上不兼容时自动回退不复权(复权日线优先走东财/akshare/Ashare)。
- 妙想/部分东财接口在某些网络下可能受代理影响失败,均有容错不影响主流程。

## 八、常见问题

- **首次无分析结果**:检查 `.env` 的 `DEEPSEEK_API_KEY`。
- **定时任务不跑**:确认 jobs_hub 常驻;任务开关是否开启(默认仅核心任务开)。
- **持仓/成交/推荐看不到历史**:变动与盈亏闭环依赖 PG(`USE_POSTGRES=true`);SQLite 模式部分历史不可用。
- **PDF 中文乱码/失败**:装中文字体(见"安装")。
- **行情拿不到**:多为外部 host 网络/代理问题,取数链有兜底;可设 `TDX_SERVERS` 指定可连的通达信服务器。
